HTML/CSS Cheatsheet

HTML

Basic HTML Structure

Tag Attribute Code Example Description
<meta> charset 
<meta charset="utf-8" />
 Specifies the character encoding for the document.
name="description" <meta name="description" content="A guide on HTML" /> Provides a short description for search engines and social media previews
name="keywords" <meta name="keywords" content="HTML, CSS, web development" /> Supplies keywords for SEO purposes, though it's less commonly used today.
name="author" <meta name="author" content="Jane Doe" /> Defines the author of the webpage.
name="viewport" <meta name="viewport" content="width=device-width, initial-scale=1.0" /> Controls the page layout on mobile devices to improve responsiveness.
http-equiv="refresh" <meta http-equiv="refresh" content="30" /> Sets the page to refresh at intervals, here every 30 seconds.
name="theme-color" <meta name="theme-color" content="#ffffff" /> Sets the browser's toolbar color on some mobile devices.
property="og:title" <meta property="og:title" content="My Awesome Page" /> Specifies the title displayed when the page is shared on social media.
property="og:description" <meta property="og:description" content="An in-depth guide on web development" /> Provides a description when the page is shared on social media.
property="og:image" <meta property="og:image" content="https://example.com/image.jpg" /> Links an image displayed with the page preview on social media.
